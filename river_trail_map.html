Lansing River Trail Map

This map shows the Lansing River Trail, a recreational trail in Lansing, Michigan. I used data from OpenStreetMap, Lansing Open Data, Michigan GIS Open Data, and the USGS. All work was done in QGIS, with the exception of Blender, which I used to create a hillshade that adds some texture to the background of the map.
